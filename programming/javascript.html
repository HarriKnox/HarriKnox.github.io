JavaScript

JavaScript is the language of browsers as every internet browser supports it in some way. It is a lovely scripting language with the Script features of Scheme and Lua but the Java syntax of Java: first-class functions, closures, prototypes, and curly braces and semicolons everywhere! Technically speaking, JavaScript and Java have nothing in common beside the name, which was probably used as a marketing ploy in the wake of the Java Hype of the late 1990's. JavaScript is technically not just a language, but an implementation of the ECMAScript Specification. While we're being technical, JavaScript is really only run on Mozilla browsers since Mozilla owns and develops JavaScript. and all the other browsers have their own version that is compliant with ECMAScript. There are various JavaScript Engines: the first and one that is run on Mozilla browsers is SpiderMonkey
