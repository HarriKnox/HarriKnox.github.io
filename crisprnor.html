CRISPR-dCas9 NOR Gate

As you can see by the header at the very top of the page, I'm a programmer. More specifically, I'm a computer engineer (CPE) with hopes to go into processor design and manufacturing. Processors are built from logic gates, which are built from diodes and transistors. Now we can add DNA to that list. Using DNA and the CRISPR-dCas9 system, scientists have been able to construct logic circuits in a cell. These cellular digital circuits can be used for gene therapy, immune system aiding, and tissue engineering.

A Background in Logic

(If you already know how processors and logic gates work skip to the next section, Gene Editing with Cas9. This is an ELI5-like explanation.)
At the very heart of computers are ones and zeros (AKA, bits), and all a computer does is move and change these bits around in a systematic fashion. To manipulate these bits computers send them into logic gates. In an abstract sense, gates' inputs and outputs are binary (only one or zero, which are often called "high" and "low", or "on" and "off", respectively). In reality, logic gates are built using transistors and are activated with voltages that correspond to high and low (in some computers, "high" corresponds to somewhere in 2.7-3.3 V to 5 V and "low" corresponds to between 0 V and 0.5-1.5 V).
Common gates that are taught in schools are the NOT, AND, and OR gates. A NOT gate has one input and simply outputs the opposite value: if the input is high, the output is low; if the input is low, the output is high. An AND gate outputs high only if all inputs are high; otherwise it outputs low. An OR gate outputs high if any inputs are high; it outputs low only if all inputs are low.
In electronics, because of the nature of transistors, AND and OR gates are impractical, thus we prefer to use NAND and NOR gates, which are simply AND and OR gates connected directly into a NOT gate (by "impractical" I mean to say it is easier to build a NAND gate than it is to build an AND gate, since an AND gate in transistors is usually built with a NAND and a NOT). A NAND gate outputs low only if all inputs are high; otherwise it outputs high. A NOR gate outputs low if any inputs are high; it outputs high only if all inputs are low. Processors are built using billions of these gates, all of which allow the computer to move bits around and display pictures of cats to your screen.

Gene Editing with Cas9

Scientists at University of Washington are using the CRISPR-dCas9 system to edit genes to build logic gates in living yeast cells. Cas9 is a protein used in bacterial cells that clamps onto the DNA and uses an RNA "library" to check and cut off foreign DNA (DNA injected by viruses, for example). Cas9 has the ability to bind anywhere on a DNA strand, as well as the ability to perform a wide array of functions. These traits are admirable, but its cutting of the DNA is permanent and undesireable, so scientists produced an inactive version called "Dead Cas9", or dCas9, that can still clamp but can't cut.
dCas9 is then used to target specific genes to activate or repress them. When dCas9 is used to activate genes, it is bound to a promoter to increase gene transcription. When it's used to repress genes, it is bound to a repressor that makes it difficult for transcription of a gene to occur.

Using dCas9 Gene Repression for a NOR gate

Using the CRISPR-dCas9 system, scientists can modify three sequences of DNA and get three dCas9 proteins to target each one: two sequences are treated as input and one is output. The input gene sequences code for the production of a signal protein that will signal gene repression of the output dCas9. By activating either or both of the input gene sequences (that is, by binding the input dCas9 with an activator) they "produce a signal" to the output dCas9 to repress the gene production. If both of the input genes are left unactivated, the gene is unsequenced, so the output dCas9 allows transcription.
The output is repressed (low) when either or both of the inputs are active (high); likewise, the output is active (high) only when both inputs are repressed/unactive (low). By doing this the scientists created a biological NOR gate, which can be used as the basis of all logic gates. They created seven CRISPR-dCas9 NOR gates and designed them in various configurations to test their efficacy as logic gates. They connected them to build other complex logic gates (such as AND, OR, XOR) and found them to be effective in these configurations.

So...

Gene editing has already been done before, but now gene editing can be done in a reversible, dynamic, and digital fashion. Using these biological digital circuits genes can be activated or repressed based on environmental factors, the immune system can be better equipped to tackle pathogens, or tissue generation can be activated with a lower risk of causing cancer. The possibilities are endless, though it will be a while until cells can play Crysis 3.
