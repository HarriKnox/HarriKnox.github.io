CPE 102 Project

One of the major courses for CPE majors at Cal Poly is the class CSC 102: Fundamentals of Computer Science. For our class project, all class members were to use Pygame and Processing to create a grid-based world simulation. The purpose of the project was an exercise in object-oriented programming, functional programming, and simple path-finding. You can view the source here for the Pygame version and here for the Processing version.

Object-Oriented (OO) Programming

Java is a famous and ubiquitous OO language with support for functional programming. Python is a famous OO and functional language. Using these two languages my classmates and I were given different flavors of OO programming in similar contexts.
