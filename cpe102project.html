CPE 102 Project

One of the major courses for CPE majors at Cal Poly is the class CSC 102: Fundamentals of Computer Science. For our class project, all class members were to use Pygame and Processing to create a grid-based world simulation. The purpose of the project was an exercise in object-oriented programming, functional programming, and simple path-finding. You can view the source here for the Pygame version and here for the Processing version.

Object-Oriented (OO) Programming

Java is a famous and ubiquitous OO language with support for functional programming. Python is a famous OO and functional language. Using these two languages my classmates and I were given different flavors of OO programming in similar contexts. For the first part of our project, our professor provided us with dirty and repetitive Python code. The code was of a Pygame project that simulated a small, grid-based world with various moving and interacting entities. The code was mostly functional: only a few classes existed. There were helper classes that were given to us (such as Point and Ordered_list), but there were also the entity classes, which only held data and had no behavior for themselves.
