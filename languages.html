The following are the various languages I have picked up over the years of programming. My favorite language is Lua, with Java and JS coming second and third. Beyond that I have passionate disdain for a few select languages (namely Ruby and Python).

Lua

Lua is a beautiful and simple scripting language that is safe for children six and up. It is easy to learn with simple syntax and no unnecessary rubbish. Without a doubt it is my absolute favorite language. There are a few downsides to Lua, but nothing that is too major.

Java

Everyone knows what Java is: it's a ubiquitous OO language that requires you to write twenty lines to print "Hello World".

JavaScript and jQuery

JavaScript is like taking the scripting principles and prototype model of Lua and the syntax of Java to make an ugly baby that grew to its current popularity and became ruler of the internet. Aside from the syntax and a few core language specific differences (like the lack of a exponent operator in JS, native threading support in Lua, and variable declaration discrepancies), JavaScript and Lua are remarkably similar.

Python

Python is another scripting language like Lua, with a few OO features, native arbitrary precision, the continue keyword, and list comprehension, but with indentation that matters, no anonymous multi-line functions, spaces instead of tabs, confusing parentheses, and much more. Python's mantra is more-or-less "There should be one-- and preferably only one --obvious way to do it" which ultimately translates to "it's my way or the highway".

Ruby

If Python and Ruby were real humans, they would be bitter enemies. While Python prides itself on being concise and having one way to do something, Ruby prides itself on being as unnecessarily redundant as it can possibly be. Ruby's mantra is essentially "There should be multiple ways to do it". There are if and its complement unless, while and its complement until. There are lambdas, procs, and blocks, and each has its own differences. There are hashes, classes, and modules, which act like Lua tables and Python classes combined. There are five types of loops:
  • (int).times{} for iterating something a definitive number of times
  • (list/hashmap).each{} for iterating over every element of an iterable
  • for for iterating something a definitive number of times or over every element of an iterable
  • while for iterating indefinitely
  • until for iterating not indefinitely enough
I am not a fan of Ruby. Most of this may come from ignorance and my lack of experience with the language, but from what I've seen I don't particulary enjoy nor understand Ruby.

HTML and CSS

HTML and CSS are cool. No complaints.
